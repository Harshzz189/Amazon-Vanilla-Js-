# Amazon Clone (Vanilla JavaScript)

This project is a simplified **Amazon-style e-commerce clone** built using **HTML, CSS, and JavaScript** without any frameworks.
The goal of the project is to practice **DOM manipulation, localStorage usage, and dynamic UI updates** while building a real-world style application.

## Features

* Product listing page
* Add to cart functionality
* Cart quantity updates
* Order summary page
* Delivery date selection
* Price calculation with shipping
* Persistent cart using **localStorage**
* Interactive UI updates using **JavaScript**

## Technologies Used

* HTML
* CSS
* JavaScript (Vanilla JS)
* LocalStorage

## How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in your browser

No installation or dependencies are required.

## Learning Goals

This project was built to strengthen understanding of:

* JavaScript logic building
* Event handling
* DOM manipulation
* Data storage using localStorage
* Structuring a medium-sized frontend project
